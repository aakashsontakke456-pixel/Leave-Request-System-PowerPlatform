Project: Automated Leave Management System (LMS)
Role: Power Platform Developer
Tech Stack: Power Apps (Canvas), Dataverse, Power Automate, Microsoft 365

🚀 Executive Summary
Developed and deployed a modern, end-to-end Leave Management System to digitize HR workflows for over 100 branches. The application replaces manual tracking with an automated, role-based portal that handles everything from request submission to multi-stage manager approvals.

🛠️ Technical Contributions
Modern UI/UX Design: Engineered a high-contrast, "SaaS-style" interface using Glassmorphism and Neumorphism principles to ensure professional aesthetics and ease of use for branch staff.

Security & Identity: Leveraged the User() function to implement automatic login detection and profile photo integration, ensuring secure, personalized access for every employee.

Data Architecture: Architected the backend using Dataverse, implementing relational tables and complex filtering logic to show users their specific request history in real-time.

Automated Workflow Engine: Designed a Power Automate cloud flow featuring sophisticated conditional logic to process "Approve/Reject" outcomes, update database records, and trigger automated email notifications to staff.

Performance Optimization: Implemented efficient data-loading techniques, including the use of collections and the Refresh() function, to maintain high application speed during high-concurrency periods.

📈 Business Impact
Efficiency: Reduced leave processing time by eliminating paper-based requests across the entire multi-state credit society.

Transparency: Provided employees with a real-time "Ticket Journey" dashboard to track the status of their applications without needing to contact HR.

Accuracy: Minimized data entry errors by auto-populating employee details directly from the organization's Active Directory.
